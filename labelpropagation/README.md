## java-labelpropagation

Java implementation of GFHF ([Zhu and Ghahramani, 2002]).

     Iterate
      1. \hat{Y}^{(t+1)} \leftArrow D^{-1} W \hat{Y}^{(t)}
      2. \hat{Y}^{(t+1)}_l \leftArrow Y_l
     until convergence to \hat{Y}^{(\infty)}
